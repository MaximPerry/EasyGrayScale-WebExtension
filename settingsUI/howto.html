How to

This Web extension is quite easy to use. Here is a quick guide to help you get started.

Clicking the icon

Clicking the plugin's icon from the toolbar will execute it. It will inject a CSS greyscale filter in a web page.

If the plugin's icon is clicked a second time, it will remove the injected CSS and the web page will become normal again.
